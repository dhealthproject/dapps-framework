Class StatisticsCommand Abstract

description

This class defines the abstraction layer used in any statistics commands. Note that statistics commands are always stateful modules due to the extension of StatefulModule in the parent class BaseCommand.

Additionally, statistics commands also implement the CommandRunner interface from nest-commander. The run() method call is defined in BaseCommand and delegates the runtime to the method implemented in this class as runWithOptions.
