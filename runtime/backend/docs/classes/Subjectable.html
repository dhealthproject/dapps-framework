payoutState?: PayoutState
  • Defined in src/payout/concerns/Subjectable.ts:59

This property is a flag that determines whether the subject payout has been executed. An subject is considered to be completed only after the payout has been confirmed on dHealth Network.

Possible values for this field in the database are of type number and listed in PayoutState. Initially, the value will always be 0 as this corresponds to "not-started".
