discover

  • discover(options?: DiscoveryCommandOptions): Promise<void>

  • Overrides DiscoveryCommand.discover

    • Defined in src/discovery/schedulers/DiscoverAccounts/DiscoverAccounts.ts:226

    This method implements the discovery logic for this command that will find relevant subjects. Subjects in this command are accounts that have previously received a transaction from the dApp's main account.

    Discovery is done in 2 steps as described below:

    • Step 1: Reads a batch of 1000 transactions and discover addresses
    • Step 2: Find or create a corresponding document in accounts
