Class PrepareBoost5Payouts

label

PAYOUT

description

The implementation for the payout preparation scheduler. Contains source code for the execution logic of a command with name: processor:PrepareBoost5Payouts.

Notably, this command fetches accounts that have not been the subject of payouts yet and then creates transactions and signs transactions.

since

v0.4.0
