Class DappDate

description

This component displays a timestamp in UTC with optional timestamp value and uses the standard <span> tag.

You can customize the look&feel of this components with component properties.

example

Using the DappDate component with timestamp and isShowTimestamp. timestamp is an epoch timestamp in seconds. isShowTimestamp is a boolean value specifying whether timestamp should also be displayed in raw format (i.e. number). 

  <template>
    <DappDate
      timestamp=1655718418
      isShowTimestamp=true
    />
  </template>



Parameters
param timestamp

The mandatory epoch timestamp (in seconds) value.

param isShowTimestamp

The optional flag specifying whether to also display timestamp in number.

since

v0.1.0
