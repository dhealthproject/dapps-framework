Class DappAccountAvatar

description

This component displays an account avatar and depends on the value of its address the avatar will be rendered in different color.

You can customize this component using custom HTML attributes as listed below.

example

Using the DappAccountAvatar component

  <template>
    <DappAccountAvatar
      :width="123"
      :height="123"
      :address="some-address"
      :hideCaption="true"
    />
  </template>



Parameters
param address

The required account address.

param width

The optional avatar width value (defaults to 261.333).

param height

The optional avatar height value (defaults to 131.313).

param hideCaption

The optional avatar hideCaption flag (defaults to false).

since

v0.1.0
