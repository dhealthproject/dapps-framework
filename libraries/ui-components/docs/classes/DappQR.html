Class DappQR

description

This component displays an authentication QR code that can be scanned using dHealth Mobile Wallet or dHealth Signer. The purpose of this QR code is to identify a dHealth Network Account ("login").

You can customize the look&feel of this components with component properties.

Caution that the downloadName parameter must have a PNG file extension. If you omit it, the component will automatically add it at the end.

Note that this component defines 1 slot that can be used to overwrite the download button.

Slot Description
download Defines the download button markup. If no value is provided, the component uses a .


example

Using the DappQR component with variants Variants that are available with the components library can take up values as defined in Variant. 

  <template>
    <DappQR
      qrCode="QRCodeGenerator.createExportTransaction(...)"
      variant="primary"
      with-download="true"
      download-name="your_awesome_dapp_contract.png"
    />
  </template>



Parameters
param variant

The optional design variant (defaults to "primary").

param qrCode

The required QR Code instance that will be displayed (defaults to an empty object).

param withDownload

An optional boolean to determine if a download button is displayed (defaults to true).

param downloadName

An optional name for the downloaded PNG binary file (defaults to "dhealth_dapp_qrcode.png").
